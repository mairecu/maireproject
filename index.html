Session 3 (Stage 1)

In this session i got to understand and work with CSS (Cascading Style Sheets). You can have several CSS files or sheet that wil be referenced in HTML and all of those are gonna be used to find out the final and visible style for your page.

A very important concept to apply is the Inheritance, wichi is the mechnism by wich properties are applied not only to a specified element, but also to its descendants. Inheritance prevents certain properties for being declared over and over again in a style sheet, allowing the software developers to wirte less code. It enhances faster-loading of web pages by users and enables the client to save money on band width and development costs.

Session 1 (Stage 2)

A Computer is a machine that can be programmed and this program (computer program) is a sequence of very precise steps that tell the computer what to do.

Python is a high level language that acts like an interpreter : It runs our programs, it interperts them, excutes the programs that we wrote in Python language by running a program in a language that the computer can understand directly. That way you are going to be able of running your program using your web browser.

The reason why we needed to create specifics languages to tell computers what to do, and not recycling our own ancients language, is because our native languages are too ambiguous and requiere too many words and explanation to define things (verbosity) and the computers need to interpret exactly what the programmer says and for that reason the language has to do that in a precise amount of steps.

The Python grammar is based in the Backus-Naur Form.

Non-terminal -- Replacement (Derivation)

The code must match the correct grammar.

Python grammar for arithmetic expressions

-Expression.... Expression Operator Expression
-Expression.... Number
-Operator.... +, *, etc
-Number.... 0, 1, etc
-Expression.... (Expression)

Variables

In Python, a variable is an assigment statement (Name=Expression). We can use variables to create a name and use that name to reffer to a variable. Once we define a variable we can change the value, and when we use that name again, it reffers to the new value.

The equal (=) sign does not have the same meaning in Python that in arithmetic. In Python it's equivalent to an assigment.

Strings

A String is a sequence of characters sorrounded by quotes. ("This is a String"). If you dont use the quotes, python interprets it as a variable, so you're going to get error, because it's not a define variable.

Functions

A function is a block of organized, reusable code that is used to perform a single, related action. they are used to avoid repetition by grouping common operations.

Functions can be break down in inputs, defined by def, and ouputs, stablished by return.
